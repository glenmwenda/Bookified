## 🚀 Overview
**Bookified** is an AI-powered learning platform that transforms static reading into an interactive experience.  
Upload any PDF, choose a voice, and start a natural conversation with your book in real time.

Designed for readers, students, and curious minds who want to understand content faster through discussion rather than passive reading.

---

## ✨ Features
| Feature | Description |
|--------|-------------|
| 📄 **Upload Any PDF** | Instantly turn your books or documents into conversational knowledge |
| 🎙 **Voice Conversations** | Speak with your book in real time using natural AI voice |
| 🎧 **Custom Voices** | Choose from multiple voice personalities for your AI |
| ⚡ **Fast & Responsive** | Built for smooth performance and real-time interaction |
| 🔐 **Secure Authentication** | User accounts powered by Clerk |
| 💳 **Subscription Tiers** | Built-in monetization with scalable subscription plans |
